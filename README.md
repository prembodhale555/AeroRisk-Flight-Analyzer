# AeroRisk Flight Safety Risk Analyzer

AeroRisk is a flight safety risk analysis system where the core decision-making logic is implemented using C++ algorithms.

## Technologies Used
- C++
- Node.js
- React.js
- MongoDB

## Features
- Flight risk score calculation
- Risk level classification (Low, Medium, High, Critical)
- User login/signup
- MongoDB data storage
- Interactive dashboard UI

## Project Structure
- cpp-engine → C++ algorithm logic
- backend → Node.js API & database
- frontend → React UI

## How It Works
1. User inputs flight parameters
2. System calculates risk score using C++
3. Risk level is determined
4. Result is displayed and stored
